# Sales-Inventory-SQL-Queries
This repository contains solutions to a series of SQL Queries shared for tackling. I used my local postgres database locally on DBeaver, a universal database management engine.

## Assignment Overview
The assignemnt required the students to:
- Create the assignment schema
- Create the necessary tables; customers, products, sales and inventory.
- Insert data into the various tables

## Tables
1. customers c

2. products p 

3. sales s 

4. inventory i
